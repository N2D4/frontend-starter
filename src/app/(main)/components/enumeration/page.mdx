# {metadata.title}

An enumeration or list, similar to HTML's `<ol>` or `<ul>` elements.

## Props \{#props}

The same as `<ol>` or `<ul>`, as well as:

- `type`: The type of enumeration. Can be `numbered`, `bulleted`, or `checklist`.
